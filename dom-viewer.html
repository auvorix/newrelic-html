Live tree of the Document Object Model of this page — generated in real time by the browser.

What is the DOM?
The browser reads your HTML file and builds this tree of objects in memory. JavaScript interacts with these objects — not the original HTML file.

Legend

Element node — an HTML tag
Attribute — name of an attribute
Attribute value — value in quotes
Text node — visible text content
Comment — HTML comment
Counter — number of child nodes
Tips:
▶ Click any row to expand it.
Every node with children shows a count badge.
